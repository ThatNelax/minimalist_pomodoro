# ⏳ Minimalist Pomodoro

A clean, simple, and distraction-free Pomodoro timer built with **Flutter**. Designed to help you stay focused and manage your time effectively without unnecessary features.

## ✨ Features
- **Clean UI:** No distractions, just your timer.
- **Cross-Platform:** Runs on Android, iOS, Web, and Desktop.
- **Customizable:** Easily adjust focus and break intervals.
- **Lightweight:** Fast performance and small app size.
